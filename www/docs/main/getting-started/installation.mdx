The <a href="https://www.npmjs.com/package/@h4ad/serverless-adapter">Serverless Adapter package</a> has everything you need to start.

## Requirements

Current versions supported by this library are 12.x and 14.x, we have not tested on other versions.

## Installing

First, inside your project, run the following command:

<Tabs>
<TabItem value="npm" label="NPM" default>

```bash
  npm run i --save @h4ad/serverless-adapter
```

</TabItem>

<TabItem value="yarn" label="Yarn">

```bash
yarn add @h4ad/serverless-adapter
```

</TabItem>
</Tabs>
